üîß Troubleshooting

Common Issues & Solutions

Q: Notes don't appear on startup

A: Make sure you had notes open when you last closed the app. Only open notes are restored.

Q: Preview doesn't show

A: Make sure you hover for a full 2 seconds. Move mouse away and try again.

Q: File won't open

A: The file may have been moved or deleted. Remove it and add the new location.

Q: Can't see the note window

A: The note might be off-screen. Close and reopen the app to reset positions.

Q: How do I disable auto-startup?

A: Open Task Manager ‚Üí Startup tab ‚Üí Disable "DesktopStickyNote"
